# remote 相关

##基本指令
```bash
#列出已经存在的远程分支
$ git remote 

# 查看远程服务器地址和仓库名称 列出详细信息
$ git remote -v | --verbose

# 查看远程服务器仓库状态
$ git remote show origin            
# 添加远程仓库地址
$ git remote add origin git@github:wswenyue/demo.git       
#修改远程地址
$ git remote set-url origin git@github.com:wswenyue/demo 

# 删除远程创库地址
$ git remote rm 

```
